This is the order of this labb task 2
1) create virtual env: python3 -m venv myenv
2) activate the virtualenv: source myenv/bin/activate
3) install django: pip install django
        or run: requirements.txt (django is there)
4) check if django is working: python -m django --version
5) create a new project: django-admin startproject somename
5.1) go to the project folder: cd somename
6) create a superuser: python manage.py createsuperuser
7) run server: python manage.py runserver
8) go to: http://127.0.0.1:8000/admin/login/?next=/admin/
